  - require 'pry'
  - binding.pry
  - <% binding.pry %>
Why use pry?
  - Check what's inside your code
  - Check the value of a variable in view
  - Check to my routes
  - test out your logic
  - check form params
  - exit will keep the server running
  - exit! will kill the server
